### All Applications
*  Improvements
     * Converted Help files to Markdown.
     * Improved Help files' rendering and formatting.
     * Improved security.
     * Updated libraries.
     * Added support for Markdown file type.
### Processing Environment
*  Fixes
     * Regex Extractor
       * Fixed Inline Mode not retaining attributes.
       * Fixed Entity Relationship Finder not finding relationships with regex results that had replacement values.
     * Term Lists
       * Fixed refresh button for term lists based on a file. File will be reloaded.
     * XML Concatenator
       * Added "Group Siblings" option to concatenate values with a common parent.
       * Added "Sort" option to sort values before concatenating.
     * Non-Lexical Extractor
       * Fixed excluding numbers in phrases with hyphenated words, such as "COVID-19" and "SA-245".
*  Improvements
     * Removed Entity List Tagger
     * Added double-click support to "Add from file" in Term List selection control.
     * Non-Lexical Extractor
       * Changed "Exclude Numbers in Phrases" to "Include Numbers in Phrases", with OFF being the default state.
       * Removed "Extract Phrases" option.
     * List Maker
       * Added "Open List" button.
### Installers
   * Installers with license/kill date are now called "Trial" installers.
### Visualization Environment
*  Improvements
     * Optimized image manager loading to improve startup and project loading times.
*  Fixes
     * Fixed Concatenated Strings not returning values when any of the fields are empty/missing.
  *  Hierarchy View
     * Fixed "Search - Multicolored Nodes" not selecting anything.
  *  All Views
     * Optimized selecting visuals from search results
